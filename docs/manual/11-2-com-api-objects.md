# 11.1.1.2. Felix COM API Objects

Below is a list of the objects provided by the COM API, with links to their documentation.

*App*
The Felix application
*App2*
An extended version of the Felix application object, with a more advanced API
*Memories*
A collection of *Memory* objects
*Memory*
A translation memory or glossary
*SearchResult*
A result of a translation search
*SearchResults*
A collection of *SearchResult* objects
*Record*
A translation or glossary record
*Records*
A collection of *Record* objects
*MemoryWindow*
Represents the Felix memory or glossary window
