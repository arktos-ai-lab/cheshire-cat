# 9. Using Felix from PowerPoint

For a quick introduction to using Felix from Microsoft® Word, see the *Quick-start Tutorial Using MS PowerPoint*.

This section describes the various parts of the interface for Microsoft PowerPoint, and how to use them.

There are two “modes” in the Word interface: *Translation Mode* and *Review Mode*. When Microsoft PowerPoint first launches after installing Felix, the interface is in Translation Mode.
