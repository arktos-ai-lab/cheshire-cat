# 7. Using Felix from Word

For a quick introduction to using Felix from Microsoft® Word, see the *Quick-start Tutorial Using MS Word*.

In order to use Felix from Microsoft Word, you must select the Microsoft Interface component during installation. See *Installation* for details.

The Felix interface for Microsoft Word consists of a menu, a toolbar, and keyboard shortcuts. See the following sections for details.

There are two “modes” in the Word interface: *Translation Mode* and *Review Mode*. When Microsoft Word first launches after installing Felix, the interface is in Translation Mode.
